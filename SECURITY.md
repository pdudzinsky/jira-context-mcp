# Security Policy

## Supported Versions

This project is in early development (alpha). Security fixes are applied
to the latest release on `main`. Older versions are not maintained.

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub
issues, discussions, or pull requests.**

Instead, use GitHub's private vulnerability reporting on this repository:
[Security tab → "Report a vulnerability"][gh-pvr].

I'll acknowledge receipt within a few business days and work with you on
a fix and a disclosure timeline.

[gh-pvr]: https://github.com/pdudzinsky/jira-context-mcp/security/advisories/new
